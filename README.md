# Flappy Bird RL

A Reinforcement Learning based Flappy Bird AI project where an intelligent agent learns to play the game automatically using Deep Q-Learning techniques.

## Features
- AI learns gameplay through rewards and penalties
- Reinforcement Learning implementation
- Automated game-playing agent
- Training visualization and score tracking

## Technologies Used
- Python
- Pygame
- NumPy
- TensorFlow/PyTorch
- Reinforcement Learning

## Concepts Used
- Deep Q-Learning (DQN)
- Reward-based learning
- Exploration vs Exploitation
- Neural Networks
